# gulp4-template

gulp4 + pug + scss + TypeScript + imagemin templete

## Quick start

1. Install pug and gulp

```bash
npm install --global pug-cli
npm install --global gulp-cli
```

1. Clone this Repository

```bash
git clone https://github.com/y3s2k2/gulp4-template.git
```

1. Start developing.

```bash
cd gulp4-template
npm install
gulp
```
